# 📸 MonicaVerse
### A Modern & Responsive Image Gallery

MonicaVerse is a modern, responsive image gallery built using **HTML5, CSS3, and Vanilla JavaScript**. Designed to provide a smooth and engaging browsing experience, it combines elegant visuals with interactive features such as live search, category filtering, favorites, a full-screen lightbox, and dark mode.

## 📖 Project Overview

MonicaVerse is more than just an image gallery—it's an interactive browsing experience. Instead of simply displaying images, the application allows users to explore beautifully organized collections through category filtering, real-time search, favorites management, and an immersive lightbox viewer.

Built entirely with **HTML5, CSS3, and Vanilla JavaScript**, the project demonstrates how core frontend technologies can be used to create dynamic, responsive, and visually engaging web applications without relying on external frameworks.

Developing MonicaVerse strengthened my frontend development skills while helping me understand how thoughtful UI interactions and responsive design contribute to a better user experience.

---

## ✨ Features

- 📂 13 organized image categories
- 🖼️ 65 carefully curated images
- 🔍 Live search with real-time filtering
- ❤️ Favorites system for saving preferred images
- 🎞️ Full-screen lightbox with Previous & Next navigation
- 🌙 Dark Mode with theme persistence
- ✨ Smooth hover effects and transitions
- 📱 Fully responsive design for desktop, tablet, and mobile devices
- ⚡ Fast, clean, and interactive user experience
